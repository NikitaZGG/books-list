# Книжный магазин

React-приложение каталога книг.

## Возможности

- Главная страница с книгами
- Каталог с поиском, сортировкой и адаптивной сеткой
- Страница деталей книги
- Мобильная версия с бургер-меню и выезжающим фильтром
- Загрузка книг из открытого API (Gutendex)

## Технологии

React, React Router, Tailwind CSS, Vite, Gutendex API

## Запуск

```bash
npm install
npm run dev
```
